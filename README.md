# 📝 TaskFlow

A simple task management web app built with Vanilla JavaScript.

## ✨ Features
- Add tasks
- Delete tasks
- Persistent storage using localStorage
- Clean UI

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript
- Git & GitHub

## 🚀 Getting Started
Clone the repository and open `index.html`.
